## 🎯 Frontend Briefing – „Jarvis Tech Core UI“

### 🧠 Ziel

* Gefühl: **AI Control Center / Hacker Terminal / Iron Man Jarvis**
* User soll denken: „Ich steuere ein System – nicht nur einen Chat“

---

## 🎨 1. Design Style

### Farben

* Hintergrund: **#0b0f14 (fast schwarz)**
* Primär: **Neon Blau (#00d4ff)** oder Cyan
* Accent: **Grün (#00ff9f)** oder Rot für Errors
* Text: Weiß / leicht grau

---

## 💡 2. Layout

### Hauptstruktur:

```
[ Sidebar ]   [ Chat / Console ]
```

---

### 🧩 Sidebar (links)

* „System Status“
* aktive Module:

  * TIME
  * CALC
  * AMAZON
* kleine LEDs:

  * 🟢 aktiv
  * 🔴 error

---

### 💬 Main Area (rechts)

#### Chat = Mischung aus:

* ChatGPT
* Terminal

👉 Messages:

* Monospace Font (Coding Style)
* leichtes „Terminal Feeling“

---

## ⌨️ 3. Input Bar

* unten fixiert
* Style:

  * schwarzer Input
  * Neon Border beim Fokus
* Placeholder:

  * `> Enter command...`

---

## ⚡ 4. Animationen (WICHTIG)

* Typing Effekt (Text wird geschrieben)
* Cursor blinkt `|`
* leichte Glow Effekte
* Loading:

  * „Processing…“
  * oder „Jarvis is thinking…“

---

## 🧠 5. Response Rendering

### TEXT

* normal als Terminal Output

### LINK

* als Button:

  * `[ OPEN AMAZON ]`

### ERROR

* rot
* leicht glitch Effekt optional

---

## 🧩 6. Extra Features (macht es krank gut)

* Command History (↑ ↓ wie Terminal)
* Autocomplete (optional)
* Quick Commands Buttons

---

## 🔊 7. Sound (optional aber krank)

* kleiner „beep“ bei Antwort
* leises UI Sound Feedback

---

## 🧠 8. Micro Details

* kleine Scanlines (Hacker vibe)
* Background Grid / Noise
* Glow auf wichtigen Elementen

---

## 🧱 9. Tech Stack (für dich perfekt)

* Next.js
* Tailwind CSS
* optional:

  * Framer Motion (Animationen)

---

## 🎯 Ziel am Ende

* fühlt sich an wie:

  * Hacker Tool
  * AI Control System
  * kein „Website Chat“

---

## 🔥 Motto

> ❌ „Chat App“
> ✅ „Control Interface“
